Post-Game Onboarding Modals

Two first-time-only modals that appear on the GameOver screen, each gated independently on an FTUE flag. Both are scoped to classic mode and fire after the server confirms the capsule result.

Variant A
First Capsule — Push to Pin Book
Fires the first time a capsule is actually earned (server-confirmed via earnCapsule → earned: true). Primary CTA routes to the Pin Book so the user can crack open their first capsule. Gated by firstCapsuleShown.
First Capsule

You just earned a Capsule!

Rip it open to reveal your first pin. It lives in your Pin Book forever.

Trigger: server returns earned: true and !ftue.has("firstCapsuleShown").
Fires 800ms after game-over so the GameOver screen animates in first.
One-shot per player
Variant B
So Close — Try Again Encouragement
Fires the first time a classic game ends without earning a capsule (score < 15K, or the server rejected the award). Primary CTA instantly kicks off a new game. Gated by firstFailShown.
So close

You scored 8,420!

Hit 15K+ next time, that’s when Pin Capsules start dropping! Special tiles (4-match, 5-match) boost your score fast.

Trigger: !actuallyEarnedCapsule and !ftue.has("firstFailShown").
Fires 800ms after game-over. Only in classic mode.
One-shot per player
Both modals layer at z-[70] over the GameOver screen with a 0.85-opacity black scrim.
